Welcome to Gnu/Linux Installation Tutorial

Ang webpage na ito ay ginawa upang turuan ang isang baguhan kung pa-paano mag hanap, mag install, at mag maintain ng isang Gnu/linux operating system. Ang mga steps ay nakasulat sa filipino, mayroong din itong mga litrato, at video upang mas madaling maintindahn ng kahit na sino.

Requirements:

  • Flashdrive na may kapasidad na hindi bababa sa 4GB.
  • Internet Connection.
Paalala:
  • Kung nais mo lang masubukan ang operating system na ito, mas mabuting gumamit nalang muna ng virtual box upang hindi magdulot ng posibleng problema sa iyong system.
  • Tandaan na laging gumawa ng backup ng mga mahahalagang files sa separate na drive upang hindi magdulot ng posibleng problema.
  • Bago mag type ng mga command sa terminal ay mabuting basahin na muna ang manual.

Maraming pwedeng pagkuhanan ng mga distribution ng Gnu/linux operating system, pero mabuting aralin at maiiging piliin ang iyong unang magiging distro. Isa sa mga kilalang site ang distrowatch upang gawin ito. Sa distro watch pwede ka magbasa, mag research, at maintindihan ang mga pinagkakaiba ng mga distro. May mga link rin dito na dadalhin ka sa official website ng distribution na iyong napili. Ang isa sa mga pinaka popular na distro ay Ubuntu, Fedora, OpenSuse, Debian, at Arch Linux.

Para sa tutorial na ito, ang aking gagamitin na distribution ay Fedora. Ang dahilan kung bakit ko napili ang distro na ito ay dahil Fedora ang pinaka angkop sa ginagawa ko. Ang mga software sa Fedora ay laging updated, ang mga firmware at iba pang mga kritikal na update ay palaging bago at maraming maintainer kaya naman ito ang aking paboritong distro. Ngunit di-gaya ng ibang distro gaya ng Ubuntu, ang Fedora ay may pag-kakumplekado ng kaunti. May mga bagay na kailangan mo gawin ng manual, at hindi automatic gaya ng Ubuntu.

Ang unang kailangan na gawin ay pumunta sa official website ng Fedora, i-click ang download now para sa Fedora Workstation. at i-click at i-install ang Fedora Media Writer. Ang Fedora Media Writer ay isang bootable drive maker na ginawa para maka-pili ng spin, ma-download, at ma-write sa iyong flashdrive, I-install mo ito dahil ito ang pangunahing software na kailangan natin upang makagawa ng bootable drive.

I-open ang Fedora Media Writer na iyong dinownload. At makikita mo ang unang interface nito, mayroong tatlong pwedeng pagpilian. Ang Fedora Workstation, Fedora Server, at Custom Image. Para sa tutorial na ito ang gagamitin natin ay fedora Workstation. I-click ito at mapupunta ka sa Fedora Workstation interface kung saan ay mayroon ka makikitang button na Create Live USB, i-click mo ito.

Ngayon naman ay nasa interface ka na nag ddownload ng image ng Fedora, kailangan mo lang lagyan ng check ang check box na Write the image immediately when the download is finished. Hintayin lamang, ang Fedora Media Writer ay automatic na I-dodownload ang pinaka latest na release ng Fedora, at I-wwrite na rin ito ng automatic sa iyong flashdrive na nakasaksak sa computer.
