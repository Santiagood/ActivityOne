Welcome to Gnu/Linux Installation Tutorial

Ang webpage na ito ay ginawa upang turuan ang isang baguhan kung pa-paano mag hanap, mag install, at mag maintain ng isang Gnu/linux operating system. Ang mga steps ay nakasulat sa filipino, mayroong din itong mga litrato, at video upang mas madaling maintindahn ng kahit na sino.

Requirements:

  • Flashdrive na may kapasidad na hindi bababa sa 4GB.
  • Internet Connection.
Paalala:
  • Kung nais mo lang masubukan ang operating system na ito, mas mabuting gumamit nalang muna ng virtual box upang hindi magdulot ng posibleng problema sa iyong system.
  • Tandaan na laging gumawa ng backup ng mga mahahalagang files sa separate na drive upang hindi magdulot ng posibleng problema.
  • Bago mag type ng mga command sa terminal ay mabuting basahin na muna ang manual.

Maraming pwedeng pagkuhanan ng mga distribution ng Gnu/linux operating system, pero mabuting aralin at maiiging piliin ang iyong unang magiging distro. Isa sa mga kilalang site ang distrowatch upang gawin ito. Sa distro watch pwede ka magbasa, mag research, at maintindihan ang mga pinagkakaiba ng mga distro. May mga link rin dito na dadalhin ka sa official website ng distribution na iyong napili. Ang isa sa mga pinaka popular na distro ay Ubuntu, Fedora, OpenSuse, Debian, at Arch Linux.

Para sa tutorial na ito, ang aking gagamitin na distribution ay Fedora. Ang dahilan kung bakit ko napili ang distro na ito ay dahil Fedora ang pinaka angkop sa ginagawa ko. Ang mga software sa Fedora ay laging updated, ang mga firmware at iba pang mga kritikal na update ay palaging bago at maraming maintainer kaya naman ito ang aking paboritong distro. Ngunit di-gaya ng ibang distro gaya ng Ubuntu, ang Fedora ay may pag-kakumplekado ng kaunti. May mga bagay na kailangan mo gawin ng manual, at hindi automatic gaya ng Ubuntu.

Ang unang kailangan na gawin ay pumunta sa official website ng Fedora, i-click ang download now para sa Fedora Workstation. at i-click at i-install ang Fedora Media Writer. Ang Fedora Media Writer ay isang bootable drive maker na ginawa para maka-pili ng spin, ma-download, at ma-write sa iyong flashdrive, I-install mo ito dahil ito ang pangunahing software na kailangan natin upang makagawa ng bootable drive.

I-open ang Fedora Media Writer na iyong dinownload. At makikita mo ang unang interface nito, mayroong tatlong pwedeng pagpilian. Ang Fedora Workstation, Fedora Server, at Custom Image. Para sa tutorial na ito ang gagamitin natin ay fedora Workstation. I-click ito at mapupunta ka sa Fedora Workstation interface kung saan ay mayroon ka makikitang button na Create Live USB, i-click mo ito.

Ngayon naman ay nasa interface ka na nag ddownload ng image ng Fedora, kailangan mo lang lagyan ng check ang check box na Write the image immediately when the download is finished. Hintayin lamang, ang Fedora Media Writer ay automatic na I-dodownload ang pinaka latest na release ng Fedora, at I-wwrite na rin ito ng automatic sa iyong flashdrive na nakasaksak sa computer.

Ngayon na mayroon ka nang bootable drive na may lamang Fedora, isaksak mo ang flashdrive sa usb port ng iyong computer. I-restart ang computer, at pumunta sa bios. pagkatapos ay ilagay sa unahang boot order ang iyong flashdrive. I-save at mag-exit.

Ngayon, pag boot ng fedora ay makikita mo na may dalawang pwedeng pag-piliian; ang Start Fedora-Workstation-Live 32, at ang Test this media & start Fedora-Workstation-Live 32. Kung kampante ka na hindi corrupted at palyado ang bootable drive mo piliin mo ang nauna, ngunit mas maigi na piliin ang pangalawa upang maiwasan na corrupted ang ma-install sa iyong computer at hindi magdulot ng problema. Hintayin lamang na mag-boot sa main screen ang operating system.

At ngayon kung nasa main screen kana, makikita mo ang isang Welcome screen na may dalawang pwedeng pagpiliian, ang Try Fedora, at ang Install to Hard Drive. Piliin ang Install to Hard Drive. Sa Installation Page, bubungad ang piliian ng language, pumili ka ng naaangkop sa iyong nais gamitin. Pagtapos mapupunta ka sa Installation Summary, pindutin ang Keyboard na logo sa screen, piliin ang naaangkop sa keyboard layout na meron ka, at pagtapos ay pindutin ang done button.

Pagtapos mamili ng layout, pindutin naman ang logo ng hard-drive. Lagyan ng check ang hard-drive na nais mo pag-installan ng operating system, at pagtapos ay pindutin ang done button. Hintayin na mag sync in ang mga pinili, kapag wala ng warning maari na pindutin ang Begin Installation button

Matagal ang pag-install kaya naman dapat na maging mapagpasensya. Kapag tapos na ang pag install maari na pindutin ang Finish Installation button. I-restart ang iyong computer, ngunit wag mo na munang bunutin ang flashdrive na iyong sinaksak.

Pag balik, bubungad sayo ang isang Welcome screen. i-next lang, at fill-upan ang mga dapat na sagutin. Pag tapos ay may lalabas na help page, kung saan pwede mo matutunan kung paano gamitin ang Gnome Desktop Environment.

Tapos na! nakapag install kana, at Welcome sa Gnu/Linux Operating System.

Ngayon na tapos ka na mag install, dapat na mag update ka na ng system mo upang maging updated at ma-install ang ibang pang dependencies na kailangan ng iyong bagong operating system.

Pindutin ang windows key sa keyboard, o command key kung naka mac. I-search ang Terminal. Kapag nasa Terminal na kailangan mo I-type ang mga command na ito:

sudo dnf update

Pag tapos i-enter ay i-type mo naman ang iyong password. pagtapos ay hintayin, dahil mag s-syncronize pa muna ang iyong computer sa repositories ng fedora. Kapag tapos na may mag pprompt sayo kung nais mo i-install, at i-update ang mga sumusunod na application. I-type ang "y", at hintayin.

Ang RPM Fusion ay isang software repositories na hindi pinayagan ng fedora/ redhat na isama sa default repositories nila, dahil sa ang mga software na laman nito ay hindi isang Free Software, o ang mga license ng mga software dito ay proprietary. Ngunit mahalagang i-install ito dahil nandito ang ilan sa mga mahahalagang software na kailangan mo upang maging smooth at updated ang iyong system, kagaya ng latest Nvidia graphics driver, at Multimedia Codecs.

Upang maintall ang RPM fusion kailangan mo pumunta sa Fedora Docs, at i-copy & paste ang command, o i-type sa Terminal ang mga command na ito:

sudo dnf install https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm && sudo dnf install https://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm

Ang multimedia codecs ay isa sa pinaka mahalagang i-install pagtapos na magkaroon ka na ng connection sa RPM Fusion repositories. By default ang Fedora ay hindi nakakapag play ng mga videos, at music maliban sa may format na .mkv at .mp3. Kaya naman kung mayroon kang mga .mp4 na videos, at ibang format ng mga music ay hindi ito gagana.

Upang mainstall ang mga multimedia codecs na kailangan mo pwede ka pumunta sa Fedora Docs, o kaya naman ay i-type ang command na ito:

sudo dnf install gstreamer1-plugins-{bad-\*,good-\*,base} gstreamer1-plugin-openh264 gstreamer1-libav --exclude=gstreamer1-plugins-bad-free-devel lame\* --exclude=lame-devel && sudo dnf group upgrade --with-optional Multimedia

Kung ikaw ay naka-laptop, ang isa sa pinaka unang naiisip mo ay kung ma-ooptimize mo ba ang battery ng laptop mo ng kagaya sa windows, kung ngayon ay naka Gnu/Linux ka na? Ang sagot ay Oo, especially kung naka Thinkpad na laptop ka. Ang sagot sa optimization ng battery para sa mga laptop sa Gnu/Linux ay TLP.

Sa website ng TLP pindutin sa kaliwa, ang drop-down na may Fedora. Basahin ang documention at sundan ang mga kailangang gawin, o di kaya naman i-type ang command na ito sa Terminal:

sudo dnf install tlp tlp-rdw

Upang umipekto ang optimization na ito, kailangan mo i-restart ang iyong computer kapag tapos na mainstall nito. Kapag tapos nito ay automatic nang mag ooptimize ang TLP sa iyong laptop, kada bubuksan mo ito.
